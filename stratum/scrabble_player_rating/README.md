# Scrabble Player Rating

Sempipes optimization experiment for scrabble player rating prediction using Kaggle data.

## Quick Start

```bash
git clone git@github.com:e-strauss/reproducibility.git
cd stratum/scrabble_player_rating

# Sets up uv, venv, downloads data, and runs optimization
./run.sh
```

## Prerequisites

- `uv` (installed automatically by `run.sh` if missing)
- Kaggle CLI configured with credentials in `~/.kaggle/kaggle.json` (for data download)
  - Get your API credentials from https://www.kaggle.com/settings
  - Place them in `~/.kaggle/kaggle.json` with format: `{"username":"your_username","key":"your_key"}`
